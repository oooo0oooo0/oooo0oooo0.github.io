eza

https://github.com/eza-community/eza

eza 是一个现代的替代品，用于显示文件列表的命令行程序 ls，它提供了更多的功能和更好的默认设置。

安装：brew install eza

使用：eza

tldr

https://github.com/tldr-pages/tldr

tldr 是一个命令行工具，用于显示命令的简短说明。

安装：brew install tldr

使用：tldr command

https://github.com/yinzhenyu-su/weekly-git-summary
